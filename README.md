# Chimage Issues Tracker

Welcome to the issue tracker for [Chimage](https://modrinth.com/mod/chimage) - a Minecraft mod that lets you paste, preview, and share images directly in the in-game chat.

I'm using this repository just to track bugs, feature requests, and suggestions for the mod.

## Found a Bug?

Before opening a new bug report, please take a quick look at the [existing issues](https://github.com/KRWCLASSIC/Chimage-Issues/issues) to make sure it hasn't been reported yet.

When you open a new issue, please try to include:
- **Minecraft Version** (e.g., 1.20.1)
- **Mod Loader** (Fabric or Forge, and the version)
- **Chimage Version** (e.g., 1.0.0)
- **What happened** (and what you expected to happen)
- **Steps to reproduce** (how can I trigger the bug myself?)
- **Logs/Crash Reports** (if you have them, upload via [Pastebin](https://pastebin.com/) or [mclo.gs](https://mclo.gs/))

## Have a Feature Request?

I'm always open to ideas to improve the mod. 
1. Check the [existing issues](https://github.com/KRWCLASSIC/Chimage-Issues/issues) first.
2. Open a new issue and describe your idea.

## Need Help?

If you just need help setting up the mod, configuring the server, or anything like that, check the main [Chimage page](https://modrinth.com/mod/chimage) first. 

Thanks for helping out!
